Getting started

To get started you first have to head to GetBootstrap.com and click on "Documentation". From the documentation tab, you can either copy the CSS stylesheet link, the JavaScript link or a basic HTML document that has both included as well. This is the easiest option for getting everything that bootstrap has to offer.

Using Bootstrap

There are a couple ways of using Bootstrap. You can use classes in your HTML to add styles to your website. The Bootstrap documentation page has all theinfo on which classes will do what. Everything from colors to centering can be made using nothing but classes.

The Second way to use Bootstrap is to add pre-built components into your HTML. Components cover a wide variety of features that you might want. From Alerts to Carousels to Nav Bars, the components make flushing out your website super easy. To use a component, naviagate to it's tab in the documentation. From there you are given instructions on how to make one yourself but you also get theoption to just copy the pre-built HTML directly into your website. This is an easy way to make hight quality layouts very quickly!

Utilities

Having trouble getting something to look right? You don't necessarily need to code some custom CSS when you have Bootstrap utilities! Utilities add a bunch of basic CSS properties using only bootstrap. Everything from Clearfix to Position to margins and padding! Once again navigate to the The Bootstrap documentation page and click on "Utilities". From there you will see a list of all the properties you can use. Much like components, utilities can be made by either using the proper class or by copying their HTML directly into your code!
